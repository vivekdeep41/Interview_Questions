Cloudtrail : CloudTrail is a service that captures information about every request sent to the Amazon Route 53 API. CloudTrail saves log files of these requests to an Amazon S3 bucket. You can use information in the CloudTrail log files to determine which requests were sent to Amazon Route 53, the IP address that the request was sent from, who sent the request, when it was sent, and more.
